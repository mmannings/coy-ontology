## Modeling Trade in Value Added (TiVA) 2021 indicators

The TiVA indicators represents global flows and services. It has four views [1].

 1. The origin of value added,
 2. Exporters,
 3. Importers,
 4. Final consumers.

Each of these views can be represented using tables [1]. To express TiVA 2021 indicators as an ontology, we modeled each view as a tree structure.
